\doxysubsection{Detailed Description}
Заголовочный файл c описанием класса Таблица группы, содержит упорядоченную таблицу студентов определенной категории (Младшекурсник или Старшекурсник). Определяет характеристики группы студентов 

Файл содержит класс Таблица учащихся.\+Таблица включает студентов определенных типов, индекс группы, категорию студентов, максимальное количество студентов в группе, максимаьлное количество оценок студентов, а также курс стужентов
